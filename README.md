## Projects

### [MIMS — Make Idea Make Sense](https://github.com/zhengminjie1981/mims)
An AI agent that guides non-technical users through software design via structured dialogue. Transforms vague ideas into domain models, requirement documents, and interactive prototypes — no coding required. Built on the FBS (Function-Behavior-Structure) modeling framework with a multi-agent architecture.

### [Skills](https://github.com/zhengminjie1981/skills)
A collection of specialized AI skills for Claude Code, covering database management, document-driven development, knowledge indexing, presentation generation, and Socratic reasoning. Documentation-first design with progressive loading.
